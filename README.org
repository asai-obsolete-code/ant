* Ant  - 

** Usage

#+BEGIN_SRC lisp
(require :ant)
(main)
#+END_SRC

| key                | description        |
|--------------------+--------------------|
| S                  | Start/Stop running |
| R                  | Reset              |
| up/down arrow keys | move               |
| Mouse1 Drag        | move               |
| +/-                | Zoom up/down       |
| Mouse2 Drag        | put walls          |
| Mouse3 Drag        | put ant's food     |
|                    |                    |
